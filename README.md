roomba-bot
==========

# Description

   This bot roams on the table(5x5) following the given commands
   Node: This bot does not clean your room!

## Usage

	Check out this repository
	$ git clone ...
	$ cd roomba-bot/bin
	$ chmod +x ./roomba_walk
	

## Test

To run test simply run rspec:

	$ rspec
