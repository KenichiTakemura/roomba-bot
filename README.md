# Roomba-bot

   This bot roams on the table following the given commands
   Node: This bot does not clean your room!

## Installation

Add this line to your application's Gemfile:

    gem 'roomba-bot'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install roomba-bot
